{# components/tag.html — single source of truth for inline status chips. Usage (the macro is registered on env.globals, so no import is needed): {{ tag("Strong", kind="severity") }} {{ tag("PASS", kind="status") }} {{ tag(level, kind="cert_level") }} {{ tag(value, kind="metric") }} {{ tag("anything") }} {# default kind is "auto" #} Renders …. The base `.tag` plus `.tag--` is the canonical convention.
